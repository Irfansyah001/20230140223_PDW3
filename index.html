CSS (Cascading Style Sheets) adalah bahasa pemrograman yang digunakan untuk memberikan gaya atau memperindah tampilan konten web. Dasar-dasar CSS menjelaskan apa saja yang diperlukan untuk memulai membuat konten web. CSS dapat mengatur bagaimana membuat sebuah teks menjadi warna merah, membuat sebuah konten muncul ditempat tertentu dengan tata letak yang dapat diatur dan lain sebagainya.

CSS bukanlah bahasa pemrograman seperti HTML. Ini juga bukan bahasa markup. CSS adalah singkatan dari Cascading Style Sheets. CSS digunakan untuk membuat gaya pada elemen HTML secara selektif :)
